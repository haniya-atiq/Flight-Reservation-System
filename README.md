# Flight Reservation System

A desktop application for searching and booking flights, built with **Java Swing** and **Oracle Database** in **Eclipse IDE**.

---

## 🛠️ Technologies Used

| Tool | Purpose |
|------|---------|
| Eclipse IDE | Development environment |
| Java SE 8+ | Programming language |
| Oracle Database XE | Database |
| JDBC (ojdbc8.jar) | Database connectivity |
| Java Swing | GUI toolkit |

---

## 🚀 Features

- User login & registration
- Search available flights
- View flight details in table
- Book seats
- Password hashing (SHA-256)
- Input validation & error handling
